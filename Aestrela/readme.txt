1. Compile os arquivos

2. Para executar: java AlgoritmoAStar arquivo origem destino

   Exemplo: 
   java AlgoritmoAStar cidades.txt 1 6
   > Caminho: 1 3 5 6   Distancia: 14


Observações:

A arquivo exemplo cidades.txt tem 6 cidades, identificadas pelos números de 1 a 6.

Formato do Arquivo cidades.txt:
==================
Numero de Cidades

Latitude e longitude de cada cidade

Distancias entre as cidades
